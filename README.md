geistesblitze
=============

Python 3.4 & pyvenv
--------------------

    $ pyvenv venv
    $ source venv/bin/activate
    $ pip install -r requirements.txt
    $ ./create_all.py
    $ ./run.py

Python 2.7 & virtualenv
-----------------------

    $ virtualenv venv
    $ source venv/bin/activate
    $ pip install -r requirements.txt
    $ ./create_all.py
    $ ./run.py
